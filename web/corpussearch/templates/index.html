Check whether a piece of text — or a substantial part of it — appears in a large document corpus, using document fingerprinting.

FindMyText: Robust, Scalable Detection of Text Containment in Large Web-Crawled Corpora

Lars Henry Berge Olsen, Pierre Lison, Martin Jullum and Mark Anderson

Norwegian Computing Center, Oslo, Norway
{lhbolsen, plison, jullum, anderson}@nr.no

How it works

  1. Pick a corpus to check against.
  2. Add your text — type or paste it, load a premade example, or add the corpus-specific generic filler.
  3. Choose one or more detection algorithms and run the check.
  4. Optional: once you have results, highlight the matching passages to see exactly which parts overlap.

We extract a compact set of fingerprints from your text and look for the same fingerprints in the corpus.
